# Issues with SPAs

- Large bundle sizes (100KB+ JS, MBs with ScalaJS)
- Slow initial page load / slow builds with ScalaJS
- Poor SEO & crawler support
- Build tool complexity (npm, webpack, Vite for ScalaJS...)
- State management complexity
- Dependency hell (version conflicts, breaking updates)
- Security vulnerabilities (supply chain attacks)
- Component libraries: large, complex and suddenly unmaintained
