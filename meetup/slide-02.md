# Issues with SPAs

- Large bundle sizes (100KB+)
- Slow initial page load
- Poor SEO & crawler support
- Build tool complexity (npm, yarn, webpack, babel, etc.)
- State management complexity
- Dependency hell (version conflicts, minor/patch updates break apis)
- Security vulnerabilities (supply chain attacks)
- Component libraries: large, complex and suddenly unmaintained
- **ScalaJS/Laminar?** - Huge bundles, slow builds, still needs Vite
  - Been there, migrated away from it
